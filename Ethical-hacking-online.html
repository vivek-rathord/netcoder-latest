Milestone 1 - Duration: 2 weeks

Networking & Cyber Security Basics • 6 Live Sessions • 2 Projects • 2 Case Studies

This milestone builds a strong foundation in cyber security and networking by introducing core concepts such as network architecture, protocols, system security, and ethical hacking fundamentals. You will understand how systems communicate, how vulnerabilities arise, and how attackers think, setting the base for advanced penetration testing and security practices.

Week 1

Networking Fundamentals & Security Basics

  • Introduction to Cyber Security & Ethical Hacking
  • Types of Hackers (White Hat, Black Hat, Grey Hat)
  • Networking Basics (LAN, WAN, TCP/IP, OSI Model)
  • IP Addressing, DNS, DHCP & Ports
  • Common Network Attacks (MITM, DoS, Phishing)
Week 2

Linux & Hacking Environment

  • Introduction to Linux for Hackers
  • Kali Linux Installation & Setup
  • Linux Commands & File System Navigation
  • Setting up Virtual Lab (VirtualBox / VMware)
  • Introduction to Ethical Hacking Tools
PROJECT

Network Scanning & Vulnerability Analysis

Perform network scanning using tools like Nmap to identify open ports, services, and potential vulnerabilities. Analyze network exposure and prepare a basic security report.

Setup Ethical Hacking Lab Environment

Build your own ethical hacking lab using Kali Linux and virtual machines. Configure vulnerable systems and practice basic penetration testing in a safe environment.

Case Study: WannaCry Attack

Analyze how the WannaCry ransomware spread globally by exploiting system vulnerabilities and understand prevention techniques.

Case Study: Phishing Attack Simulation

Learn how phishing attacks work by simulating a basic phishing scenario and understanding how users are targeted and protected.

Milestone 2 - Duration: 5 weeks

Ethical Hacking & Penetration Testing • 6 Live Sessions • 2 Projects

This milestone introduces core ethical hacking techniques and penetration testing methodologies. You will learn how attackers gather information, scan systems, and identify vulnerabilities using industry-standard tools and techniques.

Week 1

Reconnaissance & Footprinting

  • Passive & active reconnaissance techniques
  • Information gathering using OSINT tools
  • Domain, DNS & IP intelligence analysis
  • Understanding attack surface mapping
  • Target profiling techniques
Week 2

Scanning & Vulnerability Analysis

  • Network scanning using Nmap
  • Identifying open ports & services
  • Vulnerability scanning tools
  • Analyzing system weaknesses
  • Introduction to exploitation

Milestone 3 - Duration: 4 weeks

Web Application Security • 12 Live Sessions

This milestone focuses on web application security and common vulnerabilities. You will learn how attackers exploit websites and how to secure applications using best security practices.

Week 1

OWASP Top 10 & SQL Injection

  • Understanding OWASP Top 10
  • SQL Injection attacks
  • Database exploitation basics
  • Input validation techniques
  • Preventing SQL vulnerabilities
Week 2

XSS, CSRF & Web Attacks

  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • Session hijacking basics
  • Client-side vulnerabilities
  • Secure coding practices

Milestone 4 - Duration: 4 weeks

Advanced Hacking Techniques • 12 Live Sessions

This milestone dives into advanced ethical hacking techniques including exploitation, wireless attacks, and social engineering strategies used in real-world cyber attacks.

Week 1

Exploitation & Metasploit Framework

  • Introduction to exploitation
  • Working with Metasploit
  • Payloads & reverse shells
  • Privilege escalation basics
  • Post-exploitation techniques
Week 2

Wireless Hacking & Social Engineering

  • Wi-Fi security vulnerabilities
  • Password cracking techniques
  • Wireless attack tools
  • Social engineering attacks
  • Human psychology in hacking

Milestone 5 - Duration: 2 weeks

Security Analyst & Career Readiness • 6 Live Sessions

This final milestone prepares you for real-world cyber security roles. You will learn threat analysis, incident response, and build your career profile for ethical hacking jobs.

Week 1

Security Monitoring & Incident Response

  • Threat detection basics
  • Incident response lifecycle
  • Log analysis & monitoring
  • SIEM introduction
  • Security reporting
Week 2

Career Preparation & Certifications

  • Resume building
  • Portfolio creation
  • Mock interviews
  • CEH & Security+ guidance
  • Cyber security career roadmap
